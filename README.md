# Curumim

Pequeno editor de texto em Rust usando Crossterm.

## Recursos

- [x] Unicode
- [ ] Tabs

### 0.1

    - [x] Abrir arquivos do sistema ou criar um novo
    - [x] Movimentação do cursor
    - [x] Undo
    - [x] Salvar arquivos
    - [ ] Número de linhas

### 0.2

    - [ ] Ferramenta de pesquisa
    - [ ] Syntax Highlight
